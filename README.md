# VoxAI-Intelligent-Voice-Assistant

VoxAI is a real-time AI Voice Assistant that combines speech recognition, NLP-based intent detection, and task automation into an interactive web application.
The assistant uses TF-IDF vectorization and cosine similarity to classify spoken user commands into predefined intents and execute corresponding actions.

🧠 Core Features
* Real-time Speech Recognition
* NLP Intent Classification (TF-IDF + Cosine Similarity)
* Intent Confidence Score Display
* Web Automation (Google & YouTube)
* Wikipedia Knowledge Retrieval
* Expression-Based Calculator
* Text-to-Speech (TTS) Engine
* Conversation History Tracking
* Context-Based Dynamic Image Rendering

🏗 System Workflow
1. Speech Input via Microphone
2. Speech-to-Text Conversion
3. Text Preprocessing (Regex Cleaning)
4. TF-IDF Vectorization
5. Cosine Similarity-Based Intent Matching
6. Action Execution
7. Text-to-Speech Response
8. Streamlit UI Rendering

🛠 Tech Stack
* Frontend: Streamlit
* Speech Recognition: SpeechRecognition
* Text-to-Speech: pyttsx3
* NLP: NLTK
* ML Techniques:
  - TF-IDF
  - Cosine Similarity
* Knowledge Retrieval: Wikipedia API
* Development Environment: Visual Studio Code
* Language: Python
